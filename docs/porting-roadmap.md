## Validation strategy

1. For each milestone, pick **sectors** from C# `TestApp` / telnet flows and compare **counts and strings** (names, spectral types).
2. Keep a small table of **expected values** in `galaxy_test` or a future unit test runner.

## Risk areas

- **Shift signedness:** C# `>>` on signed vs unsigned; port uses explicit unsigned types where the original uses `ulong`.
- **Floating point:** Minimize float work until `StarSystem` port; prefer matching C# `float`/`double` usage when touching the same formulas.
